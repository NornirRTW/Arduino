Install Arduino IDE (windows version, all default seettings)

Install ESP32 board support 
	- https://randomnerdtutorials.com/installing-the-esp32-board-in-arduino-ide-windows-instructions

Download the esp32-ttgo repository 
	- https://github.com/osresearch/esp32-ttgo

Install the Nornir_ESP32 library

Choose the "TTGO LoRa32-OLED V1" board


Select your own WiFi name/password

Upload to the board

Send a string to RTW
	- http://demo.rtw.no/stringPush
